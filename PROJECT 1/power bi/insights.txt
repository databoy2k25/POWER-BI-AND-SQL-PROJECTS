# Sales Insights Dashboard

## Overview
This Power BI project provides a comprehensive analysis of sales performance for Atliq Hardware. The dashboard highlights key metrics, market trends, and customer insights, enabling strategic decision-making.

## Key Features

### 1. Sales Performance Metrics
- **Total Revenue**: $84.82M  
- **Total Quantity Sold**: 195.82K units  
- **Average Order Value**: $6.77K  

### 2. Market Insights
- **Top Markets by Revenue**:  
  Delhi NCR leads with $47M in revenue, followed by Mumbai and Ahmedabad.  
- **Top Markets by Quantity Sold**:  
  Delhi NCR again dominates with 82K units, followed by Mumbai (36K) and Kochi (20K).  

### 3. Temporal Analysis
- A year filter slider (2017-2020) and a month selection provide dynamic time-based analysis.  
- Trendline displays total revenue fluctuations across selected years.  

### 4. Detailed Breakdown
- **Top 5 Markets**:  
  - Revenue: Delhi NCR ($0.52B), Mumbai ($0.15B)  
  - Average Order Value: Lucknow ($30K), Bengaluru ($27K)  
  - Quantity Sold: Delhi NCR (0.99M units), Mumbai (0.38M units)  

- **Top 5 Customers**:  
  - Revenue: Electricalsara Stores ($0.41B)  
  - Average Order Value: Leader ($127K)  

- **Top 5 Products by Revenue**:

## Value Proposition
This dashboard serves as a vital tool for stakeholders to:  
- Identify high-performing markets, products, and customers.  
- Monitor revenue trends and order volumes across various time periods.  
- Optimize sales strategies based on data-driven insights.  

## Technology Stack
- **Data Visualization**: Power BI  
- **Data Source**: Sales and customer data for Atliq Hardware  
- **Key Features**: Interactive filters, bar charts, trendline visualizations, and dynamic slicers.  

## Objective
This project demonstrates expertise in transforming raw data into actionable insights, leveraging Power BI's capabilities to create interactive, visually appealing dashboards that support strategic decision-making.
